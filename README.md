# ML Grade Predictor

## Overview
A simple machine learning script that predicts student grades using:
- Linear Regression
- Decision Tree
- Random Forest

## What this project demonstrates
- Data cleaning (missing values, duplicates)
- Feature encoding
- Train/test splitting
- Model evaluation (RMSE, R²)
- Basic visualization

## How to run
pip install -r requirements.txt  
python script.py  

## Output
- Cleaned dataset
- Model metrics
- Prediction plots
- Feature importance chart
